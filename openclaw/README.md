# OpenClaw - Self-Hosted AI Assistant

OpenClaw is a self-hosted AI assistant platform that integrates with multiple messaging platforms including WhatsApp, Telegram, Slack, Discord, and more.

## Features

- **Multi-channel messaging integration** - Connect with WhatsApp, Telegram, Slack, Discord, and more
- **Browser automation** - Built-in headless browser for web interaction
- **Multiple AI providers** - Anthropic, OpenAI, OpenRouter, Gemini, xAI, Groq, Mistral, AWS Bedrock, Ollama, and more
- **Extensible plugin system** - Discord, memory-core, and other plugins
- **Gateway API** - Built-in gateway for external integrations
