# OpenClaw Template

OpenClaw is a self-hosted AI assistant platform that integrates with multiple messaging platforms including WhatsApp, Telegram, Slack, Discord, Google Chat, Signal, iMessage, and more.

## Features

- **Multi-Channel Integration**: Connect to numerous messaging platforms
- **Local Gateway Architecture**: WebSocket-based control plane running on your device
- **Voice Interaction**: ElevenLabs-powered always-on speech interaction
- **Browser Automation**: Control browsers and interact with web applications
- **Extensible Tools**: File operations, cron scheduling, and skills registry
- **Security-First**: Pairing-based access control for direct messages
